# SkillConnect

SkillConnect is an open-source full-stack service marketplace app built with Android and Spring Boot.

Backend Repo : https://github.com/Ashvin424/skill_connect_backend

## Features
- User authentication (JWT)
- Service posting & browsing
- Bookings
- Ratings & reviews
- Image uploads
- Notifications (work in progress)

## Tech Stack
- Android (Java)
- Spring Boot
- PostgreSQL
- Firebase
- AWS
